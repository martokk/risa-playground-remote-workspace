WWAA LoRA Caption Workflow for ComfyUI
======================================================

A custom workflow designed for ComfyUI that will generate captions of your images specifically for LoRA training.
You can add the trigger word that is prepend to the caption generated by either of the two ways: Florence 2 or Clip Interrogator.

You can easily switch between the two, all the details on how to best use this workflow are in the video tutorial

https://youtu.be/5WiwquvlyHM

Designed for ComfyUI

Download this and more workflows as https://weirdwonderfulai.art/

Installation
======================================================
1. Launch ComfyUI (make sure you update to latest)
2. Drag the provide PNG or JSON file into ComfyUI
3. Install any missing Custom Node (that show up in red) using ComfyUI Manager.
   Quick tutorial on installing ComfyUI Manager https://youtu.be/ltMbJaZoifQ
